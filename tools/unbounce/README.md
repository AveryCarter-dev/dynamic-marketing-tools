# Unbounce

## Description
Landing page builder with dynamic text replacement and personalized content capabilities. Unbounce allows marketers to create, test, and optimize landing pages with dynamic content that can be personalized based on visitor data.

## Link
[Unbounce](https://unbounce.com)

## Type
- SaaS

## Dynamic Functionality
- Landing Page

## Key Features
- Dynamic text replacement
- Personalized content based on visitor data
- A/B testing capabilities
- Drag-and-drop page builder
- Conversion tracking and analytics
- Integration with marketing tools
- API access for automation
